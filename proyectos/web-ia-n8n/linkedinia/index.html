Asistente IA LinkedIn

Consultas de RR.HH. con datos en tiempo real de LinkedIn (RAG con Cohere + FAISS)

Descripción del proyecto

Asistente IA LinkedIn es un asistente conversacional para Recursos Humanos que responde preguntas utilizando datos en tiempo real de perfiles, empleos y compañías en LinkedIn. Implementa un enfoque de Generación Aumentada por Recuperación (RAG), combinando Cohere (embeddings y modelo de lenguaje) con FAISS (búsqueda semántica) y LangChain para el enrutamiento/razonamiento. La interfaz está construida con Streamlit.

Flujo general: se consultan datos vía API de LinkedIn, se generan embeddings con Cohere, se indexan en FAISS, y ante cada pregunta del usuario se recupera el contexto más relevante para generar una respuesta precisa.

*En el repo se incluyen instrucciones para credenciales (.env con RAPIDAPI_KEY y COHERE_API_KEY) y ejecución con streamlit run LinkedInIARAGTarnowski.py.

Tecnologías

Interfaz:
Streamlit
RAG:
Cohere (embeddings + LLM) + FAISS
Orquestación:
LangChain
Datos en tiempo real:
LinkedIn API vía RapidAPI
