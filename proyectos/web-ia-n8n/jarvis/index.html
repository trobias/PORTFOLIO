Jarvis — Google Workspace WhatsApp Chatbot

Asistente automatizado de Gmail, Calendar y Drive vía WhatsApp

Descripción del proyecto

Jarvis es un asistente inteligente que conecta Google Workspace con WhatsApp para ejecutar tareas reales de productividad: enviar y leer correos, crear eventos en Calendar, listar archivos de Drive y responder con IA.

El sistema utiliza n8n como motor de automatización, integrado con Evolution API para la comunicación por WhatsApp. Desde allí, los usuarios pueden escribir comandos o mensajes naturales (“agendar reunión”, “leer mi último mail”, “buscar archivo en Drive”), y el flujo ejecuta las acciones correspondientes usando las APIs oficiales de Gmail, Drive y Calendar.

Además, incluye un módulo de IA conversacional (basado en Gemini / OpenAI) que interpreta mensajes ambiguos y permite consultas más naturales. Jarvis es un ejemplo de cómo un chatbot puede actuar como hub personal de trabajo dentro de WhatsApp.

Funcionamiento resumido

  • Interfaz conversacional en WhatsApp conectada a Evolution API.
  • Automatizaciones con n8n para Gmail, Calendar y Drive.
  • Interpretación inteligente con IA (Gemini / OpenAI).
  • Respuestas dinámicas: confirmación de correos enviados, eventos creados, archivos compartidos, etc.
  • Diseño modular y extensible mediante contenedores Docker.

Tecnologías

Automatización:
n8n (nodos Code, HTTP y Google APIs)
Mensajería:
Evolution API (WhatsApp)
Integraciones:
Gmail, Google Drive, Google Calendar
IA:
Gemini / OpenAI API
Infraestructura:
Docker Compose
Base de datos:
Postgres (persistencia de flujos y usuarios)
Lenguaje:
JavaScript (n8n nodes)
